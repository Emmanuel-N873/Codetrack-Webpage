# CodeTrack - Emmanuel Nwauzobilom

A responsive landing page for CodeTrack, a habit-building app for people learning to code.

## Features

- Hero section with a coding-activity visual
- Social-proof statistics bar
- Feature cards with hover lift effects
- Responsive “How it works” cards
- Testimonial and call-to-action sections
- Responsive footer
- Mobile navigation menu with open/close behavior and keyboard support

## Project structure

```text
CodeTack.html   Main landing-page markup
style.css       Page styling, responsive layouts, and animations
script.js       Mobile navigation menu behavior
Image/          Logo, activity visuals, and design-reference images
wip2.html       Standalone hero-section experiment
```

## Run locally

No build step or dependencies are required.

1. Open `CodeTack.html` in a modern web browser.
2. Resize the browser below 600px to see the mobile menu icon.
3. Select the menu icon to open or close navigation links and authentication buttons.

## Notes

- The page uses the Boxicons CDN for the mobile menu and close icons.
- Image paths are relative to the project’s `Image` folder.
- The mobile menu can also be closed by pressing `Escape` or selecting a navigation link.
